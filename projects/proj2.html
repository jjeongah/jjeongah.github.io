Relation Extraction(RE)

November 4, 2022 - December 1, 2022
Natural Language Processing (NLP) task that predicts the attributes and relationships of words (entities) in a sentence

[Task Description]
Given an input sentence along with subject_entity and object_entity, the system should produce an output that includes the predicted relation label (pred_label) from a set of 30 possible relation classes and the associated predicted probabilities (probs) for each of those 30 classes.

[Dataset]
- number of train.csv: 32470
- number of test_data.csv: 7765 (Using the label "blind=100" to represent the hidden/unknown label)
